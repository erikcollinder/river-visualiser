River Flow Visualizations

WebGL-based particle flow simulations across procedural terrain

A collection of experiments exploring particle-based river flows using Three.js. Rivers are represented as directed acyclic graphs (DAGs) that drape over procedurally generated terrain, with tens of thousands of particles creating organic flow patterns.

River Flow Visualizer

The main interactive visualization with DAG-based river networks, terrain conformance, and particle width illusion. Includes edit mode for creating custom river networks.

Main Demo

Noise Paths

Exploration of noise-based path generation for creating natural-looking river curves without manual edge placement.

Offset Paths

Experiments with lateral particle displacement and width modulation to create the illusion of river width from individual particle trails.

Velocity Field

Velocity-based flow simulation using vector fields to guide particle movement across the terrain surface.

Curvature Speed

Particle speed variations based on path curvature, simulating faster flow in straight sections and slower flow around bends.
